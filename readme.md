## 苦戦した点

  - 画像登録のページを作ってからログインページを作ったせいか、画面遷移で混乱
  - 画面遷移を整理したところで、DBの画像登録テーブルにカラムが足りてないことに気づく
  - カラム追加してみたら、色々エラーとワーニングが出てきた
  - 何とかなったけど、遊びで設定してた乱数がちょっと悪さしているので、要修正
